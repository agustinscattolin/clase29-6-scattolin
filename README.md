# Raiz

Sitio institucional estatico para un taller de restauracion de muebles y academia online.

## Estructura

- `index.html`: contenido principal de la pagina.
- `styles.css`: estilos responsive y sistema visual.
- `script.js`: menu mobile, header y tabs del proceso.
- `assets/images/`: imagenes locales usadas por la web.

## Como abrirlo

Abrir `index.html` en el navegador. La pagina funciona como sitio estatico y puede compartirse copiando la carpeta completa.

Las tipografias Syne y Raleway se cargan desde Google Fonts cuando hay conexion a internet. Si no hay conexion, el sitio usa fuentes sans-serif del sistema.
